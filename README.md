## Supported Platforms & Installation

TrowlLord is built for **Windows**, **macOS (Universal/Apple Silicon & Intel)**, and **Linux**. 

Because this is a community project, you might have to fight with anti-malware systems.

### 🔹 Windows Installation
1. Download the latest `.exe` from the [Releases](https://github.com/mrnice/trowllord-releases/releases) page.
2. Launch the installer. 
3. If Windows SmartScreen blocks the launch with a "Windows protected your PC" popup:
   * Click **More info**.
   * Click **Run anyway**.

### 🔹 macOS Installation
1. Download the universal `.dmg` (Intel & Apple silicon).
2. Drag TrowlLord to your `Applications` folder.
3. Because the app is not signed by Apple, double-clicking it normally will show a warning that the developer cannot be verified. To bypass this:
   * **Right-click** (or Control-click) the TrowlLord icon in your Applications folder.
   * Select **Open** from the context menu.
   * Click **Open** again in the confirmation dialog. (You only need to do this once).

### 🔹 Linux Installation
1. Download the Linux build from the Releases page.
2. Extract the archive and ensure the binary has executable permissions (`chmod +x`).

## Core Features

TrowlLord replaces the stock interface with a suite of built-in quality-of-life tools, automation features, and cosmetic configurations.

### Interface & Customization
* **Dark Mode:** A darker color palette tailored for comfortable viewing during long sessions.
* **Selectable Cursors:** Choose from multiple custom mouse cursor styles.
* **Player Sprite Overrides:** Locally swap or customize player character sprites on your screen.

### Navigation & World Tools

* **Minimap:** A persistent, glanceable map overlay to help track your immediate surroundings.
* **Mapping Tool:** Dedicated tools to help chart areas, log coordinates, and navigate the world.

<img width="613" height="309" alt="Screenshot 2026-06-16 at 6 34 33 AM" src="https://github.com/user-attachments/assets/8aba61a1-69c8-4525-8f65-9412ba621713" />

### Automation & Audio

* **Macro Engine:** Write and execute both classic-style macros and advanced **Lua macros**.
* **Built-in Script Editor:** Edit, debug, and manage your Lua scripts directly inside the client.
* **Bard Studio:** Specialized tools designed for composing, managing, and playing Bard music.
* **Per-Sound Volume Mixing:** Fine-tune your audio experience by adjusting individual sound effects and volume channels independently.

### Utility & Controls

* **Account Manager:** Securely store, switch between, and manage multiple character accounts seamlessly.
* **On-Screen Hotbar:** A clickable shortcut bar to instantly trigger frequently used macros, items, or commands.
* **Gamepad Support:** Full controller mapping allowing you to play Clan Lord without relying solely on a keyboard and mouse.

## Staying Up to Date

TrowlLord is actively maintained with periodic bug fixes and feature additions. You do not need to manually check GitHub for every new release.

To check for updates within the application:
1. Open the client and navigate to **Settings**.
2. Select the **About** tab.
3. Click the **Check for Updates** button to see if a newer version is available for download.
4. **Confirm download** and
5. **Restart** the application
